HTML

Introduction

What is HTML? HTML is the structure of a webpage. It tells the browser what each part of the content means, such as headings, paragraphs, lists, sections, and tables.

Explain the head and the body tags.

The head tag contains information about the webpage that is not shown directly on the page. It usually includes the title, character set, links to stylesheets, and metadata.

The body tag contains all the visible content of the webpage, such as text, images, buttons, lists, and tables.

Basic Tags

Explain the section, article, aside, header, footer tag?

header: Contains the top part of a page or section, usually a logo, title, or navigation.

section: Groups related content together in a section.

article: Represents independent content, like a blog post or news article.

aside: Contains extra or side content, like related links or notes.

footer: Contains the bottom part of a page or section, like copyright or contact info.

2. CSS

Introduction

What is CSS? CSS (Cascading Style Sheets) is a language used to style HTML elements. It controls how a webpage looks, including colors, fonts, spacing, borders, and layout.

What is the purpose of CSS? The purpose of CSS is to make webpages visually organized and attractive. It separates content from design, so HTML handles structure and CSS handles appearance.

How do you apply a CSS to an HTML (explain the 3 options)

Inline CSS

p style="color: blue;"Hello/p

Internal CSS

style p { color: blue; } /style

External CSS

link rel="stylesheet" href="styles.css"

Example styles.css:

p { color: blue; }

What is a selector? What is a CSS rule?

A selector is the part of CSS that chooses which HTML element will be styled.

A CSS rule is the complete instruction made of a selector and a declaration block.

Example:

p { color: red; font-size: 16px; }

Basic CSS

Explain:

Tag Selector: Selects all elements of a specific tag. Example: p { color: red; }

Class Selector: Selects elements with a specific class using ".". Example: .box { color: blue; }

Tag and Class Selector: Selects a specific tag with a class. Example: p.box { color: green; }

Id Selector: Selects a unique element using "#". Example: #main { color: yellow; }

What is a class? What is an id? How are they related to CSS?

A class is a reusable attribute that can be applied to multiple elements. An id is unique and used for a single element. Both are used in CSS to apply styles.

Explain the CSS Box Model? Show an example.

The box model consists of content, padding, border, and margin.

Example:

div { margin: 10px; padding: 20px; border: 5px solid red; }

Explain the media queries

Media queries are used to make webpages responsive by applying styles depending on screen size.

Example:

@media (max-width: 600px) { body { background-color: gray; } }

Layout CSS

Explain display: grid

Grid is a layout system that allows creating rows and columns to organize content.

Explain display: flex

Flexbox is a layout system used to align and distribute items in a container.
